# Money Tracker CLI tool

### What is the point of this
This CLI tool has been created for [boot dev's](https://boot.dev) Personal Project 1 course.

### How to run the project
Run the compile.sh file, this will generate a **money** executable, I have not tested how this works on operating systems other than linux, so there may be need to modify the compile file. (At least until I learn how to use cmake :D).
